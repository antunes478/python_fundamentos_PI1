Website de Compras de Produtos de Informática

Nosso website vende produtos de informática

O problema que resolvemos é proporcionar produtos com custo abaixo do mercado, oferecendo mais oportunidades para pessoas que não têm acesso a eles.

A solução para esse problema é oferecer preços mais baixos do que o mercado atual.

As tecnologias usadas para criar este website foram HTML, CSS e Python.

CRUD:

  • C: Criar promoções e criar conjuntos de peças para serem vendidos como combo.
  • R: Ler as informações detalhadas de cada produto.
  • U: Fazer atualizações diárias nos produtos e nas promoções.
  • D: Remover produtos, usuários e promoções.
